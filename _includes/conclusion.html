Conclusion and Limitations

To conclude, the performed study was very informative about the potential correlations between quotes and stock market returns and shifts. We were able to analyse Elon Musk’s quotes on a natural language level, by extracting several features from it which were linked afterwards to the stock variations and also used to build a predictive model.

However, the obtained results were not very conclusive. There are several limitations which tend to lump the models built.
The first limitation is related to the sentiment analysis. Based on the obtained scores, one could affirm that it’s not a very robust tool to use for the correlation analysis or for the prediction step. Indeed, the prediction based on the gradient boosting method, which is the best among all the trials, has an R^2 score of only ~12%. Looking for more relevant features would perhaps have led to a better score. We know though that stock prediction is a hard matter to deal with. The processed dataset could also be problematic. We filtered it using some keywords which were relevant to the case study. Maybe if we would have worked with a larger part of the dataset, i.e not filtered by a lot, we would obtain better results. Overall, the study was very interesting to discover the aspects of natural language processing for finance related matters.
